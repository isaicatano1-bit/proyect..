#Inventory Management System
#Description

This project is a simple inventory management system developed in Python. It works through a console-based menu that allows the user to manage products efficiently.

#Features

The program includes options such as adding new products, displaying the inventory, searching for a specific product, updating product quantities, and viewing general statistics.

It also allows saving the inventory data into a CSV file and loading existing data from a CSV file.

#Functionality

The system stores products with their name, price, and quantity. It can calculate total quantity, total price, identify the most expensive product, and find the product with the highest quantity.

#Usage

To use the program, the user selects an option from the menu and follows the instructions displayed on the screen.
